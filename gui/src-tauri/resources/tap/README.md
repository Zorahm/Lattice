# TAP-драйвер (вшивается в сборку)

Положи сюда **официальный подписанный** установщик `tap-windows6` от OpenVPN,
например `tap-windows-9.24.7-I601-Win10.exe`
(https://build.openvpn.net/downloads/releases/).

Зачем: backend (`lattice-client::tap`) работает с адаптером `tap-windows6`.
Команда `install_tap_driver` находит **любой `.exe`** в этой папке и запускает
его тихо (`/S`) — приложение к этому моменту уже с правами администратора.

Подписанный драйвер невозможно сгенерировать из исходников проекта — поэтому
файл кладётся сюда вручную. Без него кнопка «Установить?» вернёт ошибку
«установщик драйвера не вшит в сборку», а ручная установка драйвера по-прежнему
сработает.

> Этот `.md` — заглушка, чтобы папка существовала в гите. Сам установщик в гит
> не коммить (большой бинарь): он подтягивается на этапе сборки релиза.
